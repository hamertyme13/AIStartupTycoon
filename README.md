# 🚀 Tech Empire

A startup management simulation game built with SwiftUI.

Build products.
Hire employees.
Raise venture capital.
Research AI technologies.
Compete against rival startups.
Become a billion-dollar AI company.

## Features

✅ Employee Management
✅ Product Development
✅ Research Tree
✅ Investors
✅ Monthly Financial Reports
✅ AI Competitors
✅ Market Share
✅ Notification Center

## Built With

• SwiftUI
• Observation Framework
• Xcode 17
• Git
